# NSMessenger - Release Notes & App Store Information

## App Store Description

**NSMessenger - Real-time Team Communication**

Connect and collaborate with your team through secure, real-time messaging. NSMessenger provides instant communication with advanced features designed for modern teams.

### Key Features:
• **Real-time Messaging** - Instant message delivery with typing indicators
• **Secure Authentication** - JWT-based security with encrypted token storage
• **Contact Management** - Easy contact discovery and friend requests
• **Group Conversations** - Create and manage team conversations
• **Cross-platform** - Works on iPhone, iPad, and Mac
• **Environment Switching** - Development and production environment support
• **Modern Design** - Clean, intuitive interface with dark mode support

### Perfect for:
- Remote teams and distributed workforces
- Project collaboration and coordination
- Secure business communications
- Real-time team updates and notifications

### Privacy & Security:
- All tokens stored securely in device keychain
- End-to-end encryption for sensitive data
- No data stored on device when logged out
- Company-specific isolated environments
